# Contributing

Contributions are welcome!

!!! note "Contribution guidelines"
    Please see the repository's `CONTRIBUTING.md` for detailed guidelines and coding standards.

- GitHub: <https://github.com/your-username/vectorflow/blob/main/examples/CONTRIBUTING.md>

Typical workflow:

1. Create a virtual environment and install dev dependencies:

```bash
pip install "vectorflow[dev]"
```

1. Run tests and linters:

```bash
pytest
ruff check .
black --check .
```

1. Open a pull request on GitHub.
